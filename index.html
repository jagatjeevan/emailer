Hi ThoughtWorkers, The time has arrived to acknowledge the contribution made by each one of you in making the Thought Works journey a resounding success. It is imperative now to visit individual performances and experiences over the last year, reflect on how each of us performed as an individual contributor, and understand the relative contributions and the impact that you have created as a TWer.

Sharing the feedback is a continuous activity and we hope that you are collecting feedback on a regular basis which you can start collating. If you still have not, please start exchanging the feedback with fellow TWers you have worked with, add your self-reflections, create a summary as per the below format, and set up a review meeting with the relevant stakeholders.

Set up your goals for the next year 2021, using the goal setting template attached.
